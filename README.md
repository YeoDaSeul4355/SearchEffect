## 👋 소개

자바스크립트를 활용하여 게 이펙트를 구현한 웹 페이지를 소개합니다. 뮤직 플레이어와 테트리스 게임은 자바스크립트로 구현되어 다양한 플랫폼에서 실행 가능하며, 사용자들에게 즐거운 게임 경험과 음악 감상을 제공합니다. 
이 이펙트를 통해 javascript의 메서드 (setInterval, setTimeOut, 반복문)과 GSAP, jQuery를 학습하였습니다.
